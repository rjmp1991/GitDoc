Git Commands

How can I init Repository on GIT?

All you need to execute is the command git init on your folder

How can I get the Repository's status on GIT?

All you need to execute is the command git status on your folder

How can I create a commit on GIT?

All you need to execute is the next commands:
git add file-to-up or git -A + git commit -m "message identifier"
Also can combine the comands such as git commit -am "message identifier"

How can I create a branch on GIT?

All you need to execute is the command git branch name-branch on your folder

How can I move between branchs on GIT?

All you need to execute is the command git checkout name-branch on your folder

How can I merge two branchs on GIT?

Go to the principal branch, execute checkout command
Next execute git merge name-change-branch

How can I save remote my repo?

Go to the principal branch, execute checkout command
Next execute git remote add origin URL

How can I download changes from remote repository?

Go to the principal branch, execute checkout command
Next execute git fetch origin
Next execute git merge origin/master

How can I uploap changes to remote repository?

Go to the principal branch, execute checkout command
Next execute git push origin master
